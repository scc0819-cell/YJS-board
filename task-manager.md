# 任務管理系統

## 功能說明

本系統提供：
1. **任務看板** - 一目了然查看所有任務狀態
2. **自動化排程** - 定時自動執行任務
3. **任務鏈** - 任務完成後自動產生新任務
4. **Agent 追蹤** - 記錄每個任務使用的 Agent

## 任務狀態

- `pending` - 待處理
- `running` - 執行中
- `completed` - 已完成
- `failed` - 失敗

## 建立任務

```bash
# 手動建立任務
openclaw memory task-add --name "任務名稱" --desc "任務描述" --agent "main"
```

## 排程任務範例

### 每日早晨檢查（08:00）
- 檢查郵件
- 檢查日曆
- 檢查天氣

### 每小時心跳檢查
- 檢查重要通知
- 檢查專案狀態

### 每日晚間總結（22:00）
- 整理當日任務
- 產生明日待辦

## 任務鏈範例

```
任務 1: 檢查郵件 → 完成後 → 任務 2: 整理重要郵件 → 完成後 → 任務 3: 產生回覆草稿
```

## API 端點

- `GET /api/tasks` - 獲取所有任務
- `POST /api/tasks` - 建立新任務
- `PUT /api/tasks/:id` - 更新任務狀態
- `GET /api/cron` - 獲取排程任務
