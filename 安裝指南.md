# 🦞 OpenClaw 完整自動化工具安裝指南

## 📋 安裝步驟

### 步驟 1：執行安裝腳本

開啟終端機，執行以下命令：

```bash
sudo ~/.openclaw/workspace/scripts/install-all-tools.sh
```

系統會要求輸入 sudo 密碼，輸入後按 Enter 開始安裝。

**預計時間：** 5-10 分鐘

---

## 🎯 安裝後的功能

### 1. PDF 文件生成
```bash
# 自動將任務報告轉為 PDF
python3 << 'EOF'
from reportlab.lib.pagesizes import A4
from reportlab.pdfgen import canvas

c = canvas.Canvas("report.pdf", pagesize=A4)
c.drawString(100, 750, "任務報告")
c.save()
EOF
```

### 2. Excel 報表生成
```python
from openpyxl import Workbook

wb = Workbook()
ws = wb.active
ws['A1'] = "任務名稱"
ws['B1'] = "狀態"
ws['A2'] = "任務 1"
ws['B2'] = "已完成"
wb.save("tasks.xlsx")
```

### 3. PowerPoint 簡報
```python
from pptx import Presentation

prs = Presentation()
slide = prs.slides.add_slide(prs.slide_layouts[0])
slide.shapes.title.text = "進度報告"
prs.save("report.pptx")
```

### 4. 郵件自動處理
```python
from imap_tools import MailBox

with MailBox('imap.gmail.com').login('user@gmail.com', 'password') as mailbox:
    for msg in mailbox.fetch():
        print(msg.subject, msg.date)
```

### 5. 日曆整合
```python
from google.oauth2.credentials import Credentials
from googleapiclient.discovery import build

service = build('calendar', 'v3', credentials=creds)
events = service.events().list(calendarId='primary').execute()
```

### 6. 網頁自動化
```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto('http://localhost:8080')
    page.screenshot(path='dashboard.png')
    browser.close()
```

### 7. OCR 文字識別
```python
import pytesseract
from PIL import Image

text = pytesseract.image_to_string(Image.open('screenshot.png'), lang='chi_tra+eng')
print(text)
```

### 8. 語音功能
```python
import pyttsx3

engine = pyttsx3.init()
engine.say("任務已完成")
engine.runAndWait()
```

## 🔧 故障排除

### 問題 1: sudo 密碼錯誤
**解決方法：** 確認輸入的是正確的 sudo 密碼

### 問題 2: pip3 未找到
**解決方法：**
```bash
sudo apt install -y python3-pip
```

### 問題 3: 部分模組安裝失敗
**解決方法：** 手動安裝失敗的模組
```bash
pip3 install <模組名稱>
```

### 問題 4: Playwright Chromium 安裝失敗
**解決方法：**
```bash
playwright install chromium
```
