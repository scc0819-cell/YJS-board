# 電廠監控 v1.1 資料字典（MVP）

更新：2026-03-01 04:51 GMT+8

## 1) 來源與刷新
- 來源：AUO SunVeillance（每小時抓取）
- 主鍵：`plant_id + timestamp`
- 時區：Asia/Taipei

## 2) 欄位定義

| 欄位 | 型別 | 必填 | 說明 | 範例 |
|---|---|---|---|---|
| plant_id | string | Y | 案場唯一識別 | CH-001 |
| plant_name | string | Y | 案場名稱 | 彰化XX國中 |
| timestamp | datetime | Y | 取樣時間（整點） | 2026-03-01 04:00:00+08:00 |
| actual_gen_kwh_today | number | Y | 今日累積發電量(kWh) | 1245.6 |
| forecast_gen_kwh_today | number | Y | 今日預估發電量(無遮陰基準) | 1300.0 |
| design_offset_enabled | boolean | Y | 是否啟用設計補正 | true |
| design_offset_factor | number | Y | 設計補正倍率(0.80~1.00) | 0.93 |
| weather_icon | enum | Y | AUO 天氣圖示 | sunny |
| irr_today | number | Y | 今日日照累積 | 3.8 |
| irr_yday_slot | number | Y | 前一日同時段日照 | 4.1 |
| inverter_availability_pct | number | Y | 逆變器可用率(%) | 99.2 |
| data_gap_min | number | Y | 資料延遲分鐘數 | 5 |

## 3) 派生欄位

| 欄位 | 計算式 | 說明 |
|---|---|---|
| adj_forecast_gen_kwh_today | forecast_gen_kwh_today * design_offset_factor * weather_factor | 調整後預估發電 |
| gen_gap_pct | (actual_gen_kwh_today - adj_forecast_gen_kwh_today) / adj_forecast_gen_kwh_today * 100 | 發電落差率 |
| gen_achievement_rate_pct | actual_gen_kwh_today / adj_forecast_gen_kwh_today * 100 | 發電達成率 |
| irr_delta_pct | (irr_today - irr_yday_slot) / irr_yday_slot * 100 | 今昨日日照差 |

## 4) weather_factor 建議
- sunny / partly_sunny：1.00
- cloudy：0.95
- rainy：0.90
- unknown：1.00（保守）

## 5) 品質檢核（ETL）
1. `forecast_gen_kwh_today > 0`
2. `0.8 <= design_offset_factor <= 1.0`
3. `0 <= inverter_availability_pct <= 100`
4. `irr_today >= 0` 且 `irr_yday_slot >= 0`
5. 欄位缺失則標記 `data_quality=warning` 並列入 P3

## 6) 報表核心輸出
- 晴天下落差 >3% 場站數
- strict 模式觸發場站數
- P1/P2/P3 分布
- Top 5 落差場站（含原因）
