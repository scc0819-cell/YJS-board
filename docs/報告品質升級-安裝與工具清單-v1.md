## 一、目標
把目前輸出從「可讀」升級到「董事長等級商務質感」。

## 二、必要安裝（優先）

### 1) 字型（繁中商務）
- Noto Sans TC（全字重）
- Source Han Sans TC（思源黑體 TC，全字重）
- Inter（英文/數字）

### 2) 排版引擎
- Typst（高品質 PDF）
- WeasyPrint（HTML/CSS 轉 PDF，版型彈性高）

### 3) 簡報引擎
- Marp CLI（Markdown 轉簡報 PDF/PPTX）
- python-pptx（可編輯 PPTX）

---

## 三、建議安裝指令（WSL Ubuntu）

> 下面先列建議，是否執行由董事長確認後再跑。

### A. 系統套件
```bash
sudo apt update
sudo apt install -y fontconfig fonts-noto-cjk fonts-noto-color-emoji \
  python3-pip wkhtmltopdf
```

### B. Python 套件
```bash
pip install --user weasyprint jinja2 matplotlib seaborn pandas openpyxl python-pptx
```

### C. Node 套件（簡報）
```bash
npm i -g @marp-team/marp-cli
```

### D. Typst（若 apt 無最新版）
```bash
# 建議用官方安裝腳本或 cargo 安裝（擇一）
# cargo install typst-cli
```

---

## 四、品牌字體策略（建議）

### 中文
- 標題：Source Han Sans TC Heavy / Noto Sans TC Bold
- 內文：Source Han Sans TC Regular / Noto Sans TC Regular

### 英數
- Inter Regular / Medium

### 禁用
- 混用多套宋體/黑體造成畫面老舊
- 預設 fallback 字型（易跑版）

---

## 五、模板標準（落地）

### 董事長報告模板（3 頁）
1. 封面 + 一句結論
2. KPI 儀表 + Top 5 風險
3. 今日行動清單 + SLA

### 技術詳報模板（8~12 頁）
1. 目標與範圍
2. 資料來源與可信度
3. 指標與判斷規則
4. 異常案例
5. 改善建議與時間表

## 六、技能/流程補強

1. 建立公司色票、字級、間距規範（Design Tokens）
2. 圖表統一主題（線條粗細、色階、標籤字級）
3. 報告輸出一律模板化（禁止臨時拼湊）
4. 上線前自動 QA：字型缺失/跑版/頁碼/目錄檢查
