## 制度概述

### 核心目標

基於您提供的 4 份分析報告，我們建構了完整的員工日報管理系統，解決以下問題：

| 問題 | 影響 | 解決方案 |
|------|------|----------|
| 日報品質與實際工作脫鉤 | 員工寫得漂亮但產出低 | 增加產出驗證機制 |
| 無法驗證完成度準確性 | 可能虛報 100% | 里程碑簽核 + 信用評分 |
| 缺乏申訴機制 | 評分不公無管道反映 | 建立評審小組 |
| 未考慮案場難度差異 | 困難案場吃虧 | 難度係數調整 |
| 主觀評分占比過高 | 可能造成不公 | 降低至 7% |

### 評分維度與權重

```
月度績效 = (日報表現 × 25%) + (案場績效 × 25%) + (風險管理 × 15%) + 
           (工作態度 × 10%) + (團隊合作 × 10%) + (主管評分 × 7%) + (客戶滿意 × 8%)
```

## 績效考核制度

### 月度考核公式

```
月度績效分數 = Σ(各維度分數 × 權重) × 難度係數
```

### 各維度計算方式

#### 1. 日報表現（25%）

```
日報表現分數 = (A 級天數 × 4 + B 級天數 × 3 + C 級天數 × 2 + D 級天數 × 1) / 總天數 × 25
```

**範例：** 陳明德 2 月（20 個工作天）
- A 級：15 天 × 4 = 60 分
- B 級：5 天 × 3 = 15 分
- C 級：0 天 × 2 = 0 分
- D 級：0 天 × 1 = 0 分
- 平均：(60+15)/20 = 3.75
- 日報表現：3.75 × 25 = **93.75 分**

#### 2. 案場績效（25%）

```
案場績效分數 = Σ(案件完成度 × 案件權重) / 總案件數 × 25
```

**範例：** 李雅婷負責 5 個案件
- 陸豐國小：100% × 1.0 = 1.0
- 媽厝國小：90% × 1.0 = 0.9
- 萬合國小：75% × 1.3 = 0.975
- 長安國小：80% × 1.3 = 1.04
- 成功國小：100% × 1.0 = 1.0
- 平均：(1.0+0.9+0.975+1.04+1.0)/5 = 0.983
- 案場績效：0.983 × 100 × 25% = **24.58 分**

#### 3. 風險管理（15%）

```
風險管理分數 = (主動通報數 × 5 + 及時處理數 × 3) × 15 / 基準數
```

**範例：** 陳明德 2 月
- 主動通報高風險：3 件 × 5 = 15 分
- 及時處理：5 件 × 3 = 15 分
- 總分：30 分
- 風險管理：30/40 × 15 = **11.25 分**

#### 4. 工作態度（10%）

由系統自動計算：
- 遲交次數：-2 分/次
- 請假天數：不扣不加分
- 加班時數：+1 分/5 小時

#### 5. 團隊合作（10%）

- 跨部門協作：+2 分/次
- 知識分享：+3 分/次
- 支援同事：+1 分/次

#### 6. 主管評分（7%）

主管主觀評價（0-100 分）× 7%

#### 7. 客戶滿意（8%）

```
客戶滿意分數 = Σ(業主評分) / 案件數 × 8
```

### 案場難度係數

| 難度 | 係數 | 判定標準 | 範例 |
|------|------|----------|------|
| **簡單** | 0.8 | 500kW 以下、單一建築 | 小型屋頂型 |
| **一般** | 1.0 | 500-2000kW、2-5 棟 | 一般學校案場 |
| **困難** | 1.3 | 2000-5000kW、多棟 + 停車場 | 大型學校 + 停車場 |
| **極困難** | 1.5 | 5000kW 以上、政府標案、PPA | 台積電 PPA |

### 月度績效計算範例

**陳明德 2 月績效：**

| 維度 | 原始分數 | 權重 | 加權分數 |
|------|----------|------|----------|
| 日報表現 | 93.75 | 25% | 23.44 |
| 案場績效 | 88.0 | 25% | 22.00 |
| 風險管理 | 90.0 | 15% | 13.50 |
| 工作態度 | 95.0 | 10% | 9.50 |
| 團隊合作 | 85.0 | 10% | 8.50 |
| 主管評分 | 90.0 | 7% | 6.30 |
| 客戶滿意 | 88.0 | 8% | 7.04 |
| **總計** | | **100%** | **90.28** |

**難度係數：** 1.3（負責馬偕護專等困難案場）

**最終績效：** 90.28 × 1.3 = **117.36 分** → **A+ 級** 🏆

## Q&A

### Q1: 員工對評分不滿怎麼辦？

**A:** 可透過申訴機制：
1. 員工於 3 日內提出申訴
2. HR 收件並轉交評審小組
3. 評審小組 5 日內審議
4. 結果通知

評審小組：HR 代表 + 部門主管 + 其他部門主管

### Q2: 如何避免員工為了分數而工作？

**A:** 
- 定期檢討制度（每季度）
- 加強教育訓練
- 強調制度是工具，不是目的
- 主管適時溝通引導

### Q3: 困難案場如何認定？

**A:** 由主管會議認定，考量因素：
- 案場規模（kW）
- 複雜度（建築數量）
- 協調難度（政府/民間）
- 工期壓力

### Q4: D 級員工如何輔導？

**A:** 為期 4 週的輔導計畫：
- 第 1 週：日報格式教育訓練
- 第 2 週：工作規劃指導
- 第 3 週：風險意識培養
- 第 4 週：綜合評估

連續 2 個月 D 級：列入績效改善計畫（PIP）

### Q5: 系統何時正式上線？

**A:** 分三階段：
1. **試行期（2 週）：** 不計入績效
2. **正式實施（1 個月）：** 50% 權重
3. **全面上線：** 100% 計入績效
