## 🐛 常見問題排除

### 問題 1: Internal Server Error

**原因**: 模板檔案缺失

**解決**:
```bash
cp /home/yjsclaw/.openclaw/workspace/server/templates/components/*.html \
   /home/yjsclaw/.openclaw/workspace/daily_report_server/templates/components/
pkill -f "python3 app.py"
cd /home/yjsclaw/.openclaw/workspace/daily_report_server && python3 app.py &
```

### 問題 2: 員工名單不正確

**原因**: 資料庫未正確初始化

**解決**:
```bash
python3 /home/yjsclaw/.openclaw/workspace/daily_report_server/setup_employees_correct.py
pkill -f "python3 app.py"
cd /home/yjsclaw/.openclaw/workspace/daily_report_server && python3 app.py &
```

### 問題 3: 服務無法啟動

**檢查日誌**:
```bash
tail -50 /tmp/server.log
tail -50 /tmp/server_error.log
```

**重啟服務**:
```bash
pkill -f "python3 app.py"
cd /home/yjsclaw/.openclaw/workspace/daily_report_server && python3 app.py &
```
