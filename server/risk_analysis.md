## ⚠️ 實務風險與特殊情境（10+ 項）

### 🔴 高優先級風險

#### 1. 資料庫備份機制缺失
**情境**：伺服器故障、誤刪除、勒索病毒
**影響**：所有日報、風險、任務資料永久遺失
**建議改善**：
- [ ] 每日自動備份 SQLite DB 到 NAS
- [ ] 保留最近 30 天的備份
- [ ] 備份驗證機制（確保可還原）

#### 2. 附件檔案無病毒掃描
**情境**：員工上傳含病毒的檔案
**影響**：可能感染伺服器或下載者
**建議改善**：
- [ ] 上傳時進行副檔名白名單檢查
- [ ] 可選：整合 ClamAV 進行病毒掃描
- [ ] 限制檔案大小（目前 10MB）

#### 3. 審計日誌可被管理員刪除
**情境**：管理員刪除不利於自己的審計紀錄
**影響**：無法追溯責任，內控失效
**建議改善**：
- [ ] 審計日誌改為「只能新增，不能刪除/修改」
- [ ] 審計日誌的刪除操作本身也要被記錄
- [ ] 定期匯出審計日誌到外部儲存

#### 4. Session 劫持風險
**情境**：使用者在公共電腦登入未登出
**影響**：他人可冒充身分操作系統
**建議改善**：
- [ ] Session 時效縮短為 4-8 小時（目前 12 小時）
- [ ] 增加「記住我」選項（可選長期登入）
- [ ] 重要操作（如刪除、重設密碼）要求重新驗證

#### 5. 多人同時編輯同一筆資料
**情境**：兩個管理員同時修改同一案場進度
**影響**：後提交的覆蓋先提交的，資料遺失
**建議改善**：
- [ ] 樂觀鎖機制（檢查 updated_at）
- [ ] 衝突時提示「此資料已被修改，請重新整理」
- [ ] 顯示「誰正在編輯」

---

### 🟡 中優先級風險

#### 6. 員工離職後帳號未停用
**情境**：員工離職，但帳號仍啟用
**影響**：前員工可登入系統，資安風險
**建議改善**：
- [ ] 離職流程 checklist 包含「停用系統帳號」
- [ ] 定期（每月）檢視未登入超過 90 天的帳號
- [ ] 帳號到期日機制（可選）

#### 7. 密碼政策不夠嚴格
**情境**：員工設定弱密碼（如 12345678）
**影響**：容易被暴力破解
**建議改善**：
- [ ] 密碼需含大小寫 + 數字 + 特殊字
- [ ] 密碼長度至少 10 碼（目前 8 碼）
- [ ] 密碼歷史檢查（不能與前 3 次相同）
- [ ] 密碼 90 天強制更換（可選）

#### 8. 未提交日報的後果不明確
**情境**：員工連續多天未提交日報
**影響**：管理無法掌握進度，績效考核無依據
**建議改善**：
- [ ] 連續 3 天未提交自動通知直屬主管
- [ ] 連續 7 天未提交自動通知董事長
- [ ] 未提交天數納入績效考核系統

#### 9. 風險/任務的 Owner 離職
**情境**：任務 Owner 離職，任務無人處理
**影響**：任務逾期，專案延宕
**建議改善**：
- [ ] 帳號停用時，自動將其任務轉給直屬主管
- [ ] 提供「批量轉移任務」功能
- [ ] 顯示「無 Owner 任務」清單

#### 10. 系統升級導致資料格式不相容
**情境**：系統升級後，舊資料無法正確讀取
**影響**：歷史資料遺失或顯示錯誤
**建議改善**：
- [ ] 資料庫版本控制（migration）
- [ ] 升級前自動備份
- [ ] 提供資料轉換腳本

---

### 🟢 低優先級風險

#### 11. 時區問題
**情境**：使用者在不同時區填寫日報
**影響**：日期計算錯誤（如：台北 23:59 vs 紐約 10:59）
**建議改善**：
- [ ] 所有時間統一使用 Asia/Taipei
- [ ] 前端顯示轉換為使用者當地時間（可選）

#### 12. 閏年/閏月問題
**情境**：2/29 或農曆閏月
**影響**：日期計算錯誤
**建議改善**：
- [ ] 使用 Python 內建 datetime（已處理閏年）
- [ ] 測試邊界案例（2/28, 2/29, 3/1）

#### 13. 大量資料時的效能問題
**情境**：系統使用 3-5 年後，資料量達數萬筆
**影響**：頁面載入變慢，查詢超時
**建議改善**：
- [ ] 增加資料庫索引（case_id, employee_id, report_date）
- [ ] 分頁載入（目前部分頁面已做）
- [ ] 定期歸檔舊資料（如：3 年前的日報）

#### 14. 自然災害
**情境**：地震、火災、水災導致伺服器損毀
**影響**：系統無法使用，資料遺失
**建議改善**：
- [ ] 異地備援（NAS + 雲端備份）
- [ ] 關鍵資料列印紙本備份（可選）
