## 3️⃣ 績效管考系統

### 功能特色

**📊 多維度統計**：

| 維度 | 指標 | 權重 |
|------|------|------|
| 日報提交 | 提交率、平均延遲 | 40% |
| 任務執行 | 完成率、平均完成天數 | 40% |
| 風險管理 | 解決率、高風險數量 | 20% |

**🏆 排名系統**：
- ✅ 個人績效排名
- ✅ 團隊平均統計
- ✅ 表現優異者（Top 3）
- ✅ 需要加強者（Bottom 3）

**📝 AI 績效報告**：
- ✅ 自動生成 Markdown 報告
- ✅ 個性化評語
- ✅ 具體改進建議
- ✅ 可儲存為 PDF

### 績效計算公式

```
總分 = 日報分數 × 0.4 + 任務分數 × 0.4 + 風險分數 × 0.2

日報分數 = min(提交率，100)
任務分數 = min(完成率，100)
風險分數 = min(解決率，100) - 高風險數量 × 10
```

### 使用方式

#### API 呼叫

```bash
# 個人績效
curl "http://localhost:5000/api/performance/employee/yang_zong_wei"

# 團隊績效
curl "http://localhost:5000/api/performance/team"

# 生成績效報告
curl "http://localhost:5000/api/performance/report?employee_id=yang_zong_wei&period=monthly"
```

#### 命令列測試

```bash
# 個人報告
python3 performance_tracker.py yang_zong_wei

# 團隊報告
python3 performance_tracker.py team
```
