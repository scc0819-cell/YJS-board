# 🎨 UI/UX 優化計畫 - 昱金生能源日報系統

**版本**: 2.0  
**日期**: 2026-03-01  
**目標**: 提升使用者體驗，建立統一、專業、直覺的介面

---

## 📋 問題分析（使用者觀點）

### 當前問題

#### 1. 導航混亂 ❌

**使用者心聲**：
> 「右上角的功能選項忽左忽右，一下在這裡、一下在那裡，找不到！」

**具體問題**：
- 不同頁面導航欄位置不一致
- 按鈕排列順序不統一
- 缺乏視覺錨點

#### 2. 文字大小不一 ❌

**使用者心聲**：
> 「這個页面的字怎麼比較大？那個页面的字又比較小？眼睛很累！」

**具體問題**：
- 標題大小不統一（16px~24px 混用）
- 內文大小不一致（12px~14px 混用）
- 缺乏字級層級系統

#### 3. 後台/前台切換不明 ❌

**使用者心聲**：
> 「我現在是在前台還是後台？怎麼突然跳過來了？」

**具體問題**：
- 缺乏明確的區域標示
- 顏色主題不統一
- 缺乏麵包屑導航

#### 4. 角色權限不明 ❌

**使用者心聲**：
> 「我是主管，但我能看到什麼？不能看到什麼？」

**具體問題**：
- 缺乏角色視覺提示
- 權限界線模糊
- 功能可用性不明確

---

## ✅ 優化方案

### 1. 統一導航系統

#### 設計原則

```
┌─────────────────────────────────────────────────────────┐
│  ⚡ 昱金生能源   首頁  填寫日報  歷史  案場  任務  風險  │
│                                                         │
│                     [用戶] [通知] [登出]                 │
└─────────────────────────────────────────────────────────┘
```

#### 規範

**固定位置**：
- Logo：左上角（固定）
- 主選單：Logo 右側（固定）
- 用戶區：右上角（固定）
- 高度：64px（固定）

**固定樣式**：
- 背景：漸層藍（#1a365d → #2c5282）
- 文字：白色（#fff）
- 間距：24px（固定）

**選單項目**（依角色）：

| 角色 | 可見選單 |
|------|---------|
| 員工 | 首頁、填寫日報、歷史查詢、案場總覽、任務追蹤、風險追蹤 |
| 主管 | 員工選單 + 管理後台 |
| 管理員 | 員工選單 + 管理後台（完整權限） |
| 管理部 | 員工選單 + 人員管理（無日報查看權） |

---

### 2. 字級系統

#### 統一規範

| 用途 | 大小 | 字重 | 顏色 | 範例 |
|------|------|------|------|------|
| 主標題 | 24px | 700 | #1a202c | 頁面標題 |
| 次標題 | 18px | 600 | #2d3748 | 卡片標題 |
| 內文 | 14px | 400 | #4a5568 | 一般文字 |
| 輔助文字 | 12px | 400 | #718096 | 說明文字 |
| 按鈕 | 14px | 600 | #fff | 按鈕文字 |

#### 應用範例

```html
<!-- 頁面標題 -->
<h1 class="page-title">填寫日報</h1>
<!-- 24px, 700, #1a202c -->

<!-- 卡片標題 -->
<h2 class="card-title">📝 今日工作項目</h2>
<!-- 18px, 600, #2d3748 -->

<!-- 內文 -->
<p class="text-body">請填寫今日工作內容...</p>
<!-- 14px, 400, #4a5568 -->

<!-- 輔助說明 -->
<small class="text-muted">必填欄位，請詳細填寫</small>
<!-- 12px, 400, #718096 -->
```

### 3. 明確的區域定位

#### 麵包屑導航

**位置**：導航欄下方，內容區上方

**樣式**：
```
🏠 首頁 / 填寫日報 / 2026-03-01
```

**規範**：
- 背景：白色
- 高度：52px
- 分隔符： / （灰色）
- 當前頁面：粗體 + 藍色

#### 區域標示

**前台（員工區）**：
- 顏色主題：藍色系
- 功能：日報填寫、查詢、案場資訊
- 標示：無特殊標示（預設）

**後台（管理區）**：
- 顏色主題：深藍 + 金色邊框
- 功能：人員管理、系統設定、報表分析
- 標示：「⚙️ 管理後台」標籤

## 📱 使用者旅程優化

### 員工日常流程

#### 早上上班（8:30）

1. **登入系統**
   - 清晰登入表單
   - 記住帳號（可選）
   - 快速進入

2. **查看 AI 建議**
   - 醒目顯示新建議
   - 分類清晰（亮點/建議/支持）
   - 一鍵查看詳情

3. **填寫日報**
   - 直覺表單流程
   - 自動儲存草稿
   - 案場下拉選單

#### 下午工作（15:00）

4. **查看任務**
   - 清晰優先級標示
   - 到期日提醒
   - 一鍵更新狀態

5. **回報風險**
   - 快速表單
   - 分類清楚
   - 可上傳照片

#### 下班前（17:30）

6. **提交日報**
   - 自動檢查完整性
   - 預覽功能
   - 一鍵提交

### 主管日常流程

#### 早上（9:00）

1. **查看團隊日報**
   - 提交率統計
   - 一鍵查看未提交
   - 快速審閱

2. **查看 AI 分析**
   - 團隊風險總覽
   - 進度落後提醒
   - 資源協調建議

#### 下午（16:00）

3. **任務指派**
   - 批量指派
   - 優先級設定
   - 到期日設定

4. **風險審閱**
   - 高風險優先
   - 一鍵分派
   - 追蹤處理進度
